# Conduct of Coding Style

## 文件夹

* 全部小写字母
* 单词使用中划线连接，单词默认使用全拼.
  * 如果单词太长超过20个字符,考虑使用缩写
* 单词优先使用复数

## 文件名

* 全部小写字母
* 使用官方的推荐命名规则：
  * 组件：*.component.htm *.component.scss *.component.ts
  * 模块：*.module.ts
  * 路由: *-routing.module.ts

* 类名 ：驼峰命名，大写字母开头
* 变量名：全部小写开头
* 字符串：全部使用单引号表示
* 注释：使用 // 开头,后面一定要带上至少一个空格

## 导入模块

* 模块导入使用单引号
